## Final Release Notes

GalleryGen `v0.1.0` is the first public MVP release.

This release delivers the core workflow for turning local images into a static gallery website directly in the browser.

### Highlights

- Multi-image upload with drag and drop support
- Non-image filtering during import
- Local image preview generation in the browser
- Editable gallery title and description
- Live default gallery template preview
- Static export bundle generation
- Browser-side ZIP download with HTML, CSS, and image assets

### Scope

- This release is intentionally focused on the MVP workflow.
- There is currently one built-in template: `default`.
- Persistence, advanced metadata editing, and additional template options are not included yet.

### Export Output

The current ZIP export includes:

- `index.html`
- `styles.css`
- `images/...` for uploaded gallery images

The output is suitable for standard static hosting.

## Final Short Launch Post

Released `GalleryGen v0.1.0`.

GalleryGen is an offline-first web app that turns local images into a static gallery site directly in the browser. Import photos, edit basic metadata, preview the result live, and export a ready-to-host ZIP with no backend or account required.

This is the first public MVP, focused on a simple end-to-end workflow:

- upload images
- preview a real gallery template
- export static files

Still early, but the full core loop is now working and ready for feedback.
