# Demo Script

## 15-25 Second Recording Script

1. Start on the GalleryGen homepage with an empty state visible.
2. Drag in a small batch of images or use the file picker.
3. Briefly show the live preview updating as images appear.
4. Edit the gallery title and description.
5. Show the preview reflecting the updated metadata.
6. Click `Export ZIP`.
7. End on the downloaded ZIP file or browser download shelf.

Suggested pacing:

- 0-3s: Empty state and product title
- 3-8s: Import images
- 8-14s: Preview updates live
- 14-19s: Edit title and description
- 19-25s: Export ZIP

## 3-Screenshot Capture Checklist

1. Empty state:
   - Clean homepage
   - Uploader visible
   - No debug clutter in focus
2. Editing state:
   - Images uploaded
   - Metadata editor visible
   - Preview clearly shown
3. Export-ready state:
   - Filled title and description
   - Gallery preview populated
   - Export button visible
